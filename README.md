# Modeling High-Risk Booking Loss in Hotel Reservations

## Overview
This project analyzes hotel booking data to identify reservations with a high risk of financial loss, focusing on late cancellations and no-shows.

Using classification models and time-series analysis, the study examines how booking characteristics and temporal patterns contribute to risk and supports more effective revenue management decisions.

## Key Insights
- Risk has temporal structure (short-term persistence and seasonal patterns)
- Risk is multi-factor and not driven by a single variable
- Model performance is moderate, with Boosted Tree performing best
- Business value comes from decision strategy and threshold optimization

## Tools & Methods
- Classification models (Decision Tree, Boosted Tree, Random Forest)
- Time-Series Analysis (Seasonal ARIMA)
- Feature Engineering and Data Preparation
- Profit Matrix Optimization
